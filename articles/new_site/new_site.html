My New Website | Meu Novo Site: fanaro.io

Slowly, this website will be deactivated in favor of my new website, fanaro.io. That's a website I've entirely made from scratch — no frameworks — using only HTML, CSS, TS, and Github Pages to host it. In the future, I might add a server and a database backend like Firebase, but, for now, GH Pages is more than enough.

The whole experience will be far more minimalist and clean, I hope everyone ends up happier. The new website is completely open source, you can find everything about it here.

The reasons for the changes are based on the fact that WordPress isn't helping me anymore: I've reached a point where I believe the maintenance of a website without frameworks is smaller than the help WordPress provides. The other reasons are performance; cluttering of the webpage with useless WordPress-related stuff; and limiting my design creativity.

I've created a lot of content already. In total, there are 107 posts, besides this one. Currently, I've been able to only transcribe about 25% of those, slowly they will all end up there, don't worry, but be patient.
